前言

很长时间了，我都没有跟更新我的博客，我去做什么了呢？

在这段时间里，我都在学习 React，完成了我第一个完整的 React 项目。是一个仿照网易云音乐的项目，实现了基本的功能，音乐播放等。

心得体会

通过此次开发，我真的体会到了 React 的强大！也理解到为什么很多大厂选择 React。

React 真的给开发者带来了很高的自由度，Hooks，Redux 真的太好用了！我可以在 React 中轻松实现一些 Vue2.0 中不容易实现的逻辑。

React 哲学中有一条：All in js. 我可算深刻体会到了，Html 中可以写 JS Css 中也可以写 JS，真的太棒了。我从未体验过如此的便利。

React 的魅力真的太多，给开发者带来了很多惊喜。

我希望在以后我能深入研究 React 的源码，窥探 Facebook 程序员智慧的结晶。

数据状态管理
