js中有很多高阶函数，能在日常的代码编写工作中提高效率。下面列举一些常用js的数组高阶函数

FOO数组

var nums = [10, 5, 11, 100, 55]

filter: 过滤

返回布尔类型 可以传递三个值 item 当前的的元素，index 当前数组下标， arr 当前整个数组

//数组取偶操作
var numbers = nums.filter((item, index, arr) => {
  return item % 2 == 0
})
console.log(numbers)

map: 映射

对所有的item进行操作

//对每个item进行*10操作
var nums2 = nums.map((item) => {
  return item * 10
})
console.log(nums2)

forEach: 迭代

无返回值

//遍历数组，无返回值！ 
nums.forEach((item) => {
 console.log(item)
})

find/findIndex :查找

查找item/返还该item/itemIndex

var item = nums.find((item) => {
  return item == 11
})
console.log(item)

reduce :累加

preValue 前数据;

0:基础数据

//数组累加操作
var total = nums.reduce((preValue, item) => {
  return preValue + item
}, 0)
console.log(total)

js中很多高阶函数，我们还需继续探索。慢慢感受js的魅力！
