项目背景

评委老师好，我是面面俱到项目组的组长，下面由我来讲解该项目的主要功能。

《面面俱到》是一个提供面向市场销售的护肤产品个性化测评平台。《面面俱到》着重于用户的产品体验，对人群进行肤质分类（油皮，混油皮，敏感肌…）。针对不同的人群提供不同的皮肤护理方案。在此系统的帮助下用户可以快速找到适合自己的护肤品。

开发技术

前端：Html5 CSS JavaScript Bootstrap Vue Vuex VueRouter Axios Webpack Node.js

后端：Node.js Koa Mysql

我们已将项目部署在云端服务器121.5.114.161上，可在线体验相关功能。

模块介绍

项目由：首页，测评，种草，我的，四大模块组成。分别实现：用户登录，肤质测评，商品测评，商品种草，护肤品推荐，查询个人信息，查询评论信息等功能。其中涉及用户个性化服务的模块需要登录才能使用。那么下面由我来讲解项目的主要功能：

项目功能

登陆 肤质测评 个人信息 护肤品大全 护肤品推荐 商品页面 商品种草 评论 展示评论 种草
